Aeropus QTS – Quick Table Grid Scripts

QTS ermöglicht es dir, Tabellen und Grids schnell und einfach in HTML zu erstellen – direkt im Dokument mit <qts> oder per Inline-Skript.

Anleitung

  1. Füge einen <qts>-Block in dein HTML ein.
  2. Definiere die Gridgröße: grid h:ZEILEN;w:SPALTEN
  3. Fülle die Zellen im <qts-content>-Block. Jede Zeile endet mit ;
  4. Optional: Style bestimmte Zellen mit <qts-style> und Koordinaten h:X;w:Y
  5. Füge das Inline-Skript ein (unten auf der Seite)

Beispiel

<qts>
grid h:2;w:3

<qts-content>
[Name];[Alter];[Status];
[Luković];[27];[Online];
</qts-content>

<qts-style>
coordinate h:1;w:* { font-weight:bold; background:#111; color:white; }
coordinate h:2;w:3 { color:lime; }
</qts-style>
</qts>

Aeropus QTS – Quick Table Grid Scripts

QTS lets you quickly create tables and grids in HTML – directly in the document with <qts> or via inline script.

Instructions

  1. Add a <qts> block in your HTML.
  2. Define grid size: grid h:ROWS;w:COLUMNS
  3. Fill cells inside <qts-content>. Each row ends with ;
  4. Optional: Style specific cells with <qts-style> and coordinates h:X;w:Y
  5. Include the inline script (at the end of the page)

Example

<qts>
grid h:2;w:3

<qts-content>
[Name];[Age];[Status];
[Luković];[27];[Online];
</qts-content>

<qts-style>
coordinate h:1;w:* { font-weight:bold; background:#111; color:white; }
coordinate h:2;w:3 { color:lime; }
</qts-style>
</qts>

grid h:2;w:3 [Name];[Alter];[Status]; [Luković];[27];[Online]; coordinate h:1;w:* { font-weight:bold; background:#111; color:white; } coordinate h:2;w:3 { color:lime; }
