Aeropus QTS – Quick Table Grid Scripts

Mit QTS lassen sich Tabellen und Grids schnell und einfach definieren – direkt im HTML oder in externen .aeqts-Dateien.

Funktionsweise:

  • Definiere ein Grid mit grid h:X;w:Y (Zeilen x Spalten)
  • Fülle die Zellen im <qts-content>-Block. Jede Zeile endet mit ;
  • Optional: Style bestimmte Zellen, Zeilen oder Spalten mit <qts-style> und Koordinaten h:X;w:Y
  • Inline QTS wird automatisch beim Laden der Seite geparst
  • Externe QTS-Dateien können via QTS.load("datei.aeqts") geladen werden

Inline QTS Beispiel:

grid h:2;w:3 [Name];[Alter];[Status]; [Luković];[27];[Online]; coordinate h:1;w:* { font-weight:bold; background:#111; color:white; } coordinate h:2;w:3 { color:lime; }

Erklärung:

- Die erste Zeile [Name];[Alter];[Status]; definiert die Kopfzeile. - Jede nachfolgende Zeile enthält die Daten. - Im <qts-style>-Block wird die Kopfzeile fett und dunkel hinterlegt, die Statuszelle grün eingefärbt.
